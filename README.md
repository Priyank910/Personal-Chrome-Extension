# 🚀 Personal Chrome Extension

A custom Chrome Extension that transforms the default new tab page into a personalized productivity dashboard.

The extension combines useful daily tools such as a live clock, personalized greetings, quick-access links, notes, and entertaining content into a clean and visually engaging interface.

---

## ✨ Features

### 🕒 Live Clock

* Real-time clock updates
* Displays hours, minutes, and seconds
* Dynamic time display

### 👋 Smart Greetings

* Personalized greeting messages
* Time-based greetings
* Interactive welcome experience

### 🔗 Favorite Links

* Quick access to frequently used websites
* Personal bookmark shortcuts
* Improved browsing productivity

### 📝 Notes Section

* Save quick reminders
* Daily note-taking
* Personal productivity support

### 😂 Joke Section

* Better browsing experience

## 🚀 Installation

### Download Repository

```bash
git clone https://github.com/Priyank910/Personal-Chrome-Extension.git
```

### Open Chrome Extensions

Navigate to:

```text
chrome://extensions/
```

### Enable Developer Mode

Turn on:

```text
Developer Mode
```

### Load Extension

1. Click **Load unpacked**
2. Select the project folder
3. Open a new tab

The custom dashboard will automatically replace the default new tab page.
